# kittylib
> An simple n' lightweight web framework

## Usage
Install the package:
```
npm i kittylib -D
```
*or* use a CDN:
```js
import Kitty from 'https://cdn.skypack.dev/kittylib';
```

## Building
Clone the repo:
```
git clone https://github.com/ThinLiquid/kitty && cd kitty
```
Install dependencies and build:
```
npm i && npm run build
```
